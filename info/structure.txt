========================================
Стуркура проекта
========================================

Структура папок и файлов:

	index.php -> главная страница форума
	login.php -> страница авторизации
	signup.php -> страница регестрации
	create.php -> страница создание нового обсуждения 


	/assets
		/css - стили
			null.css -> обнуление стандартных стилей
			style.css -> стили форума
			login.css -> стли для страничка авторизации
			create.css -> стили страницы создания нового обсуждения
			button.css -> стили для кнопок

		/fonts - шрифты
			/DM-Sans - шрифты семейства DM Sans
			/Montserrat - шрифты семейства Montserrat
			fonts.css -> подключение шрифтов

		/img - изображения 

		/js
			script.js -> скрипты

	/configs
		db.php -> подключение к базе данных
		databse-sql -> база данных

	/info
		plan.txt -> план проекта
		structure.txt -> структура проекта

	/particles - части сайта 
		header.php -> верхняя часть форума
		menu.php -> боковое меню форума
		products.php -> область размещения продуктов (тема обсуждений)
		ws_person.php -> рабочая область авторизованого пользователя
